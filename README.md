# Task 1: Population Data Visualization

## Objective
To visualize the population distribution of countries using real-world data.

## Dataset
- Source: World Bank
- Indicator: Total Population (SP.POP.TOTL)
- Year: 2022

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Visualization
A horizontal bar chart showing the top 10 most populous countries in 2022.

## Key Insights
- China and India have significantly higher populations than other countries.
- Population distribution is highly skewed toward a few countries.

## Conclusion
This task demonstrates how population data can be effectively visualized using Python.
